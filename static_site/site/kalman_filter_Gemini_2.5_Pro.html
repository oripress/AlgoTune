This task solves a Kalman filtering/smoothing problem by formulating it as a convex optimization problem (Quadratic Program).

Given a linear dynamical system with process noise w and measurement noise v:
  x_{t+1} = A * x_t + B * w_t   (State dynamics)
  y_t     = C * x_t + v_t       (Measurements)

The goal is to find the state sequence x = [x_0, ..., x_N] and the noise sequences w = [w_0, ..., w_{N-1}], v = [v_0, ..., v_{N-1}] that best explain the observed measurements y = [y_0, ..., y_{N-1}], assuming a known initial state x_0.

Problem Formulation:

    minimize_{x, w, v}   sum_{t=0}^{N-1} ( ||w_t||_2^2 + tau * ||v_t||_2^2 )
    subject to           x_{t+1} = A*x_t + B*w_t,    for t = 0, ..., N-1
                         y_t     = C*x_t + v_t,       for t = 0, ..., N-1
                         x_0 = x_initial

where:
    x_t is the state vector at time t (size n).
    w_t is the process noise vector at time t (size p).
    v_t is the measurement noise vector at time t (size m).
    y_t is the measurement vector at time t (size m).
    A is the state transition matrix (n x n).
    B is the process noise input matrix (n x p).
    C is the measurement matrix (m x n).
    tau > 0 is a parameter weighting the measurement noise cost relative to the process noise cost.
    N is the number of time steps.
    x_initial is the known initial state.
    The optimization variables are the sequences x = [x_1,...,x_N], w = [w_0,...,w_{N-1}], v = [v_0,...,v_{N-1}].

This QP formulation finds the maximum a posteriori (MAP) estimate of the state trajectory assuming Gaussian noise.

import numpy as np
from scipy.linalg import pinv, solve

class Solver:
    def solve(self, problem: dict, **kwargs) -> dict:
        """
        Solves the Kalman filtering problem using a Rauch-Tung-Striebel (RTS)
        smoother. This is computationally efficient for long time horizons as
        it avoids building and solving a single large KKT system.
        """
        A = np.array(problem["A"])
        B = np.array(problem["B"])
        C = np.array(problem["C"])
        y = np.array(problem["y"])
        x_initial = np.array(problem["x_initial"])
        tau = float(problem["tau"])

        if y.ndim == 1:
            y = y.reshape(-1, 1)

        N, m = y.shape
        n = A.shape[1]
        p = B.shape[1]

        # Define noise covariances from the objective function
        # min sum(||w_t||^2) -> process noise on w is I. In state eq, this is B*I*B.T
        Q = B @ B.T
        # min sum(tau * ||v_t||^2) -> measurement noise v has cov (1/tau)*I
        R = (1.0 / tau) * np.identity(m)

        # --- Forward Pass (Kalman Filter) ---
        x_predicts = np.zeros((N + 1, n))
        P_predicts = np.zeros((N + 1, n, n))
        x_updates = np.zeros((N + 1, n))
        P_updates = np.zeros((N + 1, n, n))

        # Initialization at t=0
        x_updates[0] = x_initial
        # P_updates[0] is already np.zeros((n, n)) as x_0 is known perfectly

        I_n = np.identity(n)

        for t in range(N):
            # Prediction step
            x_pred = A @ x_updates[t]
            P_pred = A @ P_updates[t] @ A.T + Q
            
            x_predicts[t+1] = x_pred
            P_predicts[t+1] = P_pred

            # Update step
            residual = y[t] - C @ x_pred
            S = C @ P_pred @ C.T + R
            
            # Use solve for numerical stability and speed
            K = solve(S.T, C @ P_pred.T, assume_a='pos').T

            x_upd = x_pred + K @ residual
            P_upd = (I_n - K @ C) @ P_pred
            
            x_updates[t+1] = x_upd
            P_updates[t+1] = P_upd

        # --- Backward Pass (RTS Smoother) ---
        x_hat = np.zeros((N + 1, n))
        x_hat[N] = x_updates[N]

        for t in range(N - 1, -1, -1):
            P_upd_t = P_updates[t]
            P_pred_t1 = P_predicts[t+1]
            
            # Use solve for numerical stability and speed
            L = solve(P_pred_t1.T, A @ P_upd_t.T, assume_a='pos').T

            x_hat[t] = x_updates[t] + L @ (x_hat[t+1] - x_predicts[t+1])

        # --- Calculate final estimates for w and v using vectorized operations ---
        v_hat = y - (C @ x_hat[:-1].T).T
        
        B_pinv = pinv(B, rcond=1e-15)
        w_hat = (B_pinv @ (x_hat[1:].T - (A @ x_hat[:-1].T))).T

        return {
            "x_hat": x_hat.tolist(),
            "w_hat": w_hat.tolist(),
            "v_hat": v_hat.tolist(),
        }

        sparse system of linear equations (the KKT system) and solving it
        efficiently with a sparse solver.

        p = B.shape[1]
