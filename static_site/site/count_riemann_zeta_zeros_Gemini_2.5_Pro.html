`N(t) ≈ (t / (2π)) * (log(t / (2π)) - 1) + 7/8`

This formula is an asymptotic approximation. The full formula includes an error term `S(t)`:
`N(t) = (t / (2π)) * (log(t / (2π)) - 1) + 7/8 + S(t) + O(1/t)`
