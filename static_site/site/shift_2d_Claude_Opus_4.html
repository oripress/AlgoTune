@numba.njit(cache=True)
def cubic_kernel(x):
    """Cubic B-spline kernel function."""
    x = np.abs(x)
    if x <= 1:
        return (1.5 * x**3 - 2.5 * x**2 + 1)
    elif x <= 2:
        return (-0.5 * x**3 + 2.5 * x**2 - 4 * x + 2)
    else:
        return 0.0

@numba.njit(cache=True)
def shift_image_cubic(image, shift_row, shift_col):
    """Fast cubic spline interpolation for 2D image shifting."""
    rows, cols = image.shape
    output = np.zeros((rows, cols), dtype=np.float64)
    
    for i in range(rows):
        for j in range(cols):
            # Calculate source coordinates
            src_row = i - shift_row
            src_col = j - shift_col
            
            # Get integer and fractional parts
            row_int = int(np.floor(src_row))
            col_int = int(np.floor(src_col))
            row_frac = src_row - row_int
            col_frac = src_col - col_int
            
            # Accumulate weighted sum over 4x4 neighborhood
            value = 0.0
            for m in range(-1, 3):
                for n in range(-1, 3):
                    row_idx = row_int + m
                    col_idx = col_int + n
                    
                    # Check bounds (constant mode - pad with 0)
                    if 0 <= row_idx < rows and 0 <= col_idx < cols:
                        weight = cubic_kernel(m - row_frac) * cubic_kernel(n - col_frac)
                        value += image[row_idx, col_idx] * weight
            
            output[i, j] = value
    
    return output

Here are the commands available to you. Ensure you include one and only one of the following commands in each of your responses:
- `edit`: Replace a range of lines with new content in a file. This is how you can create files: if the file does not exist, it will be created. Here is an example:
```
edit
file: <file_name>
lines: <start_line>-<end_line>
---
<new_content>
---
```

The command will:
1. Delete the lines from <start_line> to <end_line> (inclusive)
2. Insert <new_content> starting at <start_line>
3. If both <start_line> and <end_line> are 0, <new_content> will be prepended to the file

Example:
edit
file: solver.py
lines: 5-7
---
def improved_function():
print("Optimized solution")
---
